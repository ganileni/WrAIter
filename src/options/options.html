How to Get a Gemini API Key

How to Get a Gemini API Key

To get a Gemini API key from Google, follow these steps:

  1. Log in to your Google account
    Make sure you are signed in to your Google account.
  2. Open Google AI Studio
    Go to Google AI Studio or search for "Google AI Studio Gemini API".
  3. Access the Gemini API section
    In AI Studio, find the "Gemini API" tab or the button labelled "Get API key in Google AI Studio".
  4. Create and copy your API key
    Click "Create API Key." Approve the terms of service if prompted. Your new API key will be displayed—copy and save it securely.

Free Plan Note:
Google offers a free tier for Gemini API usage, but it comes with rate limits. They are more than enough for personal use with WrAIter.

Official Documentation:
For more details and usage examples, visit the Gemini API documentation.

How to Get an OpenAI API Key

How to Get an OpenAI API Key

To obtain an OpenAI API key, follow these steps:

  1. Create or log in to your OpenAI account
    Visit the OpenAI Platform and sign up or log in with your credentials.
  2. Navigate to the API section
    Once logged in, click your profile icon in the top right corner and select "API Keys" or "Manage API Keys" from the dropdown menu.
  3. Create a new API key
    Click the "Create new secret key" button. You can name your key for easier management. Once generated, copy and securely store the key—OpenAI will only show it once.
  4. Add credits to your account
    If you are a new user, you may need to add payment information and purchase credits before the API key becomes active for usage.

Security Reminder:
Never share your API key publicly or expose it in client-side code. Keep it in a secure, server-side environment.

Official Documentation:
For detailed instructions and best practices, refer to the OpenAI API documentation.
